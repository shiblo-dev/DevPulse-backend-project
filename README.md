 # DevPulse

## Project Overview
DevPulse is a backend API project for managing issues/bug tracking system with authentication, roles, and issue management features.

## Live URL
https://dev-pulse-chi-one.vercel.app/

## Features
- User authentication (JWT)
- Role-based access (maintainer, contributor)
- Create, update, delete issues
- Issue status management
- Error handling middleware
- RESTful API design

## Tech Stack
- Node.js
- Express.js
- TypeScript
- PostgreSQL / Neon
- Vercel (deployment)
